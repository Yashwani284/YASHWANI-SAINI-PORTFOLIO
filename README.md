# YASHWANI-SAINI-PORTFOLIO
Personal Portfolio - Yashwani
Overview

This is a personal portfolio website built with HTML and CSS, showcasing my skills, projects, and contact information. It is designed for B.Tech CSE (AIML) students to present themselves professionally online.

Features

Responsive navigation bar with links to Home, About, Skills, Projects, and Contact sections.

Hero section with profile photo, name, and a short introduction.

About section describing educational background, interests, and goals.

Skills section displaying technical skills with progress bars.

Projects section with project cards including title, description, and a view button.

Contact section with email and a contact form.

Smooth scrolling for in-page navigation.

Technologies Used

HTML5 – Structure and content

CSS3 – Styling, layouts, gradients, hover effects

Responsive Design – Works on desktop and mobile

Folder Structure
portfolio/
│
├── index.html        # Main HTML file
├── style.css         # CSS file for styling
└── assets/           # Folder for images or other assets

How to Run

Clone or download the repository.

Open index.html in your web browser.

Ensure style.css is in the same folder or correctly linked in HTML.

Optional: Replace placeholder images and text with your personal info
